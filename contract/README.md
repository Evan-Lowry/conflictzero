# ConflictZero Compact contract

The source contract proves exact fingerprint-set disjointness for a private eight-slot firm registry and a private four-party engagement. See [`../docs/contract-architecture.md`](../docs/contract-architecture.md) for the exact statement, privacy boundary, threat model, and integration rules.

```bash
compact update 0.31.0
npm run compact
npm run test:model
```

`src/managed/conflict-zero` is generated by Compact 0.31.0. Browser integrations must use its `pureCircuits.entityFingerprint`, `pureCircuits.commitRegistry`, and `pureCircuits.commitEngagement`; JavaScript-native hashes are audit metadata, not substitutes for the Compact commitments.
